# pendor_battletime
Pendor with Battletime
Install instructions:
1. Download Pendor v3.9.5
2. Extract pendor into your modules folder
3. Rename your pendor module folder to "pendor_battletime"
4. Download the release for pendor_battletime
5. Extract the contents of pendor_battletime into your root Mount and Blade Warband folder. This will overwrite your warband .exes and the files in the pendor_battletime module folder
6. Launch using WSELoader.exe and select the pendor_battletime module
